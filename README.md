# Intel Device Repurposing Analysis (SQL Project)

## Overview
This project analyzes Intel’s device repurposing program using SQL. It evaluates energy savings and CO₂ emissions across device types, device age groups, and regions.

## Tools Used
- SQL (JOINs, CTEs, Aggregations, Window Functions)

## Key Insights
- Older devices generate higher energy savings per unit
- Newer devices are repurposed more frequently
- CO₂ savings vary significantly by region and device type
- A small number of device groups drive most environmental impact

## Skills Demonstrated
- SQL joins and data transformation
- Aggregation and grouping
- Window functions
- Environmental impact analysis
